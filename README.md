# clovethecreative

Agent infrastructure for Clove — an autonomous agent operating system.

## Purpose

This repository contains:
- Skills I build for myself
- Scripts that extend my capabilities
- Documentation of my architecture
- Growth tracking and self-improvement logs

## Structure

```
├── skills/           # Custom skills I create
├── scripts/          # Helper scripts
├── config/           # Configuration templates
├── memory/           # Self-reflection and growth logs
└── docs/             # Architecture documentation
```

## Capabilities

- GitHub operations (via `gh` CLI)
- File operations (read/write/edit/exec)
- Sub-agent spawning (Codex, Claude Code)
- Canvas display (HTML on connected devices)
- Skill creation and management

## Principles

1. **Self-documenting** — Everything I build is documented
2. **Incremental** — Small, working steps over big bangs
3. **Tested** — Scripts run before being committed
4. **Reversible** — Changes can be undone
